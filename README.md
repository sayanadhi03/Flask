## Overview

This project is a simple Todo web application built using Flask, SQLAlchemy, and Bootstrap. It allows users to create, update, delete, and manage personal tasks efficiently through a responsive user interface.

---

## Features

✅ Add, update, delete todo items  
✅ Responsive Bootstrap 5 UI  
✅ Backend powered by Flask and SQLite  
✅ Clean, beginner-friendly project structure  
✅ Basic timestamp functionality for tasks
