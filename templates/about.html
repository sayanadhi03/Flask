About MyTodo App

This simple Todo app helps you keep track of your tasks efficiently. Here's how to use it:

  • ✅ Add tasks by entering the title and description on the home page.
  • ✏️ Edit tasks by clicking the 'Update' button beside each task.
  • 🗑️ Delete tasks by clicking the 'Delete' button beside each task.
  • 🕒 The app automatically records the date and time when a task is created.

This app is built using Flask, SQLAlchemy, and Bootstrap 5.3 for a clean and responsive UI.
